Introduction

This is the documentation for GD Editor Online at muffin5671.github.io/gdeditor. Here you will find tutorials on how to use it, how I made it and more.
GD Editor is an recreation of Geometry Dash's level editor, made to run in your browser. I know there have already been websites that made the editor (like place.gd or GD Colon's recreation of the editor in a spreadsheet), but the first one only works at certain times and Colon's has limitations. As of now, the project isn't finished and I don't know if it will be. But, you never know, and miracles can happen.
